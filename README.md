# THE WORDS PROMT

AI-система генерации профессиональных промтов для переводчиков.

## Возможности

- Загрузка файлов: изображения, PDF, Word, Excel, CSV, TXT (до 10 файлов)
- Автоматический анализ типа документа, отрасли, стиля, тональности
- Поддержка глоссариев (файлы, ссылки, ручной ввод)
- Генерация промта для переводчика с учётом числовых форматов и культурной адаптации
- Генерация чек-листа самопроверки для редактора
- 15 языковых пар

## Запуск

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`

## Сборка

```bash
npm run build
```

Результат в папке `dist/`.

## API-ключ

Приложение использует Anthropic Claude API. Введите ваш API-ключ (`sk-ant-api03-...`) в интерфейсе. Ключ хранится только в сессии браузера и не передаётся на сторонние серверы.

## Технологии

- React 18 + Vite
- Mammoth.js (парсинг .docx)
- Anthropic Claude API (claude-sonnet-4)
