# THE WORDS PROMT

AI-система генерации профессиональных промтов для переводчиков.

## Возможности

- Загрузка файлов: изображения, PDF, Word, Excel, CSV, TXT (до 10 файлов)
- Автоматический анализ типа документа, отрасли, стиля, тональности
- Комментарии и пожелания для точной генерации промтов
- Поддержка глоссариев (файлы, ссылки, ручной ввод)
- Генерация промта для переводчика с учётом числовых форматов и культурной адаптации
- Генерация чек-листа самопроверки для редактора
- 15 языковых пар

## Запуск

```bash
npm install
npm run dev
```

Эта команда запускает одновременно:
- **Vite** (фронтенд) на `http://localhost:5173`
- **Express прокси** (API) на `http://localhost:3001`

Vite автоматически проксирует запросы `/api/*` на Express-сервер.

### Запуск по отдельности

```bash
npm run dev:client   # только фронтенд
npm run dev:server   # только прокси-сервер
```

## Сборка для продакшена

```bash
npm run build
```

Результат в папке `dist/`. Для продакшена запустите прокси-сервер и настройте отдачу статики из `dist/`:

```bash
npm start
```

## Архитектура

```
Браузер → localhost:5173 (Vite) → /api/* → localhost:3001 (Express) → api.anthropic.com
```

Прокси-сервер нужен потому, что Anthropic API не поддерживает прямые запросы из браузера (CORS). Express-сервер пробрасывает запросы с API-ключом на сервер Anthropic без CORS-ограничений.

## API-ключ

Введите ваш API-ключ (`sk-ant-api03-...`) в интерфейсе. Ключ передаётся через прокси напрямую в Anthropic API и нигде не сохраняется.

## Технологии

- React 18 + Vite
- Express.js (API-прокси)
- Mammoth.js (парсинг .docx)
- Anthropic Claude API (claude-sonnet-4)
